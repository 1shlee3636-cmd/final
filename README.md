# Alien Claw — 미니 3D 인형뽑기

로컬에서 간단하게 실행하려면 브라우저로 `index.html`을 열면 됩니다. 모바일에서도 동작하도록 터치 컨트롤을 제공합니다. 보안 제한 때문에 로컬 파일로 열었을 때 일부 브라우저에서 음원 재생/모델 로딩이 제한될 수 있으니 로컬 서버로 실행하세요.

파일 목록
- index.html — 메인 페이지
- styles.css — 스타일
- app.js — Three.js 기반 게임 로직

추가된 기능
- 점수와 코인: `플레이 (1코인)` 버튼을 눌러 플레이, 성공 시 점수와 코인 보상 획득
- 하이스코어 저장: 게임은 `localStorage`에 점수/코인/최고기록을 저장합니다
- GLTF 모델 로더: 상단의 입력에 GLTF/GLB URL을 넣고 `모델 불러오기`로 외부 모델을 적용 가능
- 배경음 토글: 배경음 재생 온/오프(사용자 오디오 파일을 추가하면 자동 재생 가능)
- 파티클 이펙트: 성공 시 간단한 파티클 표시

권장 실행(로컬 서버):
```bash
python -m http.server 8000
# 브라우저에서 http://localhost:8000 열기
```

다음 개선 제안
- 더 고퀄 모델과 텍스처(GLTF/GLB) 추가
- 전문 녹음된 효과음 및 BGM 교체
- 물리 엔진 통합(Ammo.js/Cannon.js)으로 현실적인 집기 동작
- UI/애니메이션 보강, 인앱 구매/광고 연동 등 상용화 요소

문제가 있거나 더 고퀄 모델·사운드를 추가하길 원하면 알려주세요.
